# 使用说明
- **使用前请核对node版本为14+**，如低于14使用`manyi-cli`可以创建项目，但是由于创建的项目是基于vite/create-react-app的，所以您创建的项目可能无法运行~

```js
npm i manyi-cli -g / yarn add manyi-cli -g /...       下载
npm uni manyi-cli -g / yarn remove manyi-cli -g /...  卸载
```

::: warning 下载不了？
如遇到下载不了，请核对您的镜像地址或换个网络
:::

## 创建vue项目

打开终端切换到指定路径，执行以下命令

```js
manyi vue
```

## 创建react项目

打开终端切换到指定路径，执行以下命令

```js
manyi react
```
